# Live Environment

Praxis boots into a shell-first live environment.

You get:

- a terminal
- the Praxis tools
- local docs in `/usr/share/doc/praxis`
- a direct install path

The core commands are:

- `praxis-help`
- `praxis-status`
- `praxis-preflight`
- `praxis-disk-report`
- `praxis-netcheck`
- `praxis-support`
- `praxis-install /mnt/praxis`
- `praxis-dev-install /mnt/praxis-dev`
- `praxis-target-check /mnt/praxis`

The goal is to keep the environment small, readable, and ready for install or
development work.
