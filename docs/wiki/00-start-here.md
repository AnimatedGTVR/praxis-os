# Start Here

Praxis is a shell-first Linux distro with a local install guide and a clear
runtime toolset.

When you boot in, you get:

- a shell
- the Praxis tools
- local documentation

The point is simple:

- inspect the system
- mount the target you want
- install the current Praxis base
- verify the target before you reboot

Useful commands:

- `praxis-help`
- `praxis-preflight`
- `praxis-status`
- `praxis-help install`
- `praxis-help qemu`
- `praxis-help commands`
